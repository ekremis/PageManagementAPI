The project is made in AngularJS, JQuery, Ajax.

Http requests (GET, POST, PUT, DELETE) used for the actions: create/edit/view/delete page, are handled through Ajax using the ASP.NET Web API.

The views are routed via AngularJS $route service (see /js/app.js). Thus, the view templates are the html files located in /partials. Each template has its own controller. Controllers are defined in js/controllers.js

User can view pages, view details of pages, edit pages, delete pages, create a new page.

Data binding is used basically in forms to bind $scope with view. The main work is done in ListCtrl controller (see js/controllers.js) where you can find all the functions doing the http requests.

The list_pages.html partial contains all views the user needs. The different sections (list pages, view details, edit a page, create a new page) are shown/hidden depending on user actions.
